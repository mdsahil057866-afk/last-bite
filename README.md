# LAST BITE 🌱

**LAST BITE** is India's first AI-powered food surplus platform. We connect NGOs with surplus food to nearby NGOs, individuals, and farmers using AI-powered Flash Sales and automated donation alerts — turning food waste into social impact.

## 🚀 Features

- **NGO Dashboard**: Restaurants, cafes, and event organizers can easily list their surplus food as Flash Sales or Donations.
- **AI-Powered Matching**: Instantly matches surplus food with nearby NGOs based on geolocation and food types.
- **Dynamic Flash Sales**: Automatic discount pricing strategies to quickly clear out excess inventory before it expires.
- **Live Tracking & OTP Delivery**: Ensure transparency and secure handoffs for food donations and purchases.
- **Real-Time Impact Analytics**: Track meals saved, CO2 prevented, and money recovered.

## 🛠️ Technology Stack

- **Frontend**: HTML5, CSS3 (Custom Design System with Glassmorphism), Vanilla JavaScript
- **Backend**: Node.js, Express.js
- **Database**: SQLite
- **Architecture**: Mobile-first, fully responsive design

## 📱 Mobile Responsive

The application is highly responsive and optimized for mobile devices with a bottom app bar navigation for seamless user experience on smartphones and tablets.

## ⚙️ How to Run Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/mdsahil057866-afk/last-bite.git
   ```
2. Navigate to the project directory:
   ```bash
   cd last-bite
   ```
3. Install dependencies for the backend:
   ```bash
   npm install
   ```
4. Start the local server:
   ```bash
   npm start
   ```
5. Open your browser and navigate to `http://localhost:5000` or simply open `index.html` (the frontend is designed to gracefully fallback to LocalStorage caching if the backend is not running!).
